# ice-ds

A neobrutalist React component library with hard shadows, bold colors, and accessible primitives.

## Features

- 30+ components
- TypeScript-first
- Tailwind CSS
- Accessible primitives
- MIT licensed

## Installation

```bash
npm install ice-ds
```

## Usage

```tsx
import { Button } from 'ice-ds'

export default function App() {
  return <Button>Click me</Button>
}
```
